# agencyErp
代理型企业资源管理
- 房产代理
- 二手车代理
- 代理装修

# 资源
## 外部资源
### 卖方资源（统称为资源）
主要是物：房源，车源，也可能是人，比如 猎头
### 买方资源：统称为客源
客源，企业
 
## 内部资源
### 员工


# 需求
## 外部资源需求：
- 新增
    - 名称
    - 属性
    - 有效性
    - 类别
    - 联系方式
- 修改
    - 修改也是一种跟进

## 内部资源需求：
- 组织管理
- 员工管理

## 员工/资源交互需求
- 资源跟进（包含打电话，写跟时，堪查（拍照，视频），带看）
- 合同管理

## 延伸需求：
- 货币管理
    - 成本管理
    - 佣金管理
    - 计算盈利
- 输出报表
